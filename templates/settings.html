Инструкция по подключению
Шаг 1: Настройте ваш веб-сервер

Перенаправьте все запросы на WAF сервер:

# Пример для Nginx
location / {
    proxy_pass http://YOUR_WAF_IP:8080;
    proxy_set_header X-API-Key "{{ user.api_key }}";
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}
Шаг 2: Проверьте подключение

Отправьте тестовый запрос через curl:

curl -H "X-API-Key: {{ user.api_key }}" http://YOUR_WAF_IP:8080/test
